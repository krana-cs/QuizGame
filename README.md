# QuizGame (Java)
A simple console-based quiz game built using Java where users answer multiple-choice questions and get a final score.

## Features
- Multiple-choice questions
- Instant answer checking
- Final score display

## Tech Used
- Java
- Scanner class
- If-else statements

## About
This project is made for practice to understand basic Java concepts like input handling, conditionals, and program flow.

## How it works
Run the program → Answer questions → Get your score at the end.
